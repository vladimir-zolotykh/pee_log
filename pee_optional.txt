Since pee-optional commit, the wclog has --pee-optional command line
argument. The log files below was created with --pee-optional
possibility in mind. For example, the log file 2024-05-24.txt has the
lines:

    1123 # default is only pee
    1123 stool # no pee

which without the --pee-optional argument would add two 'pee' tagged
record at 1123 the the db.
